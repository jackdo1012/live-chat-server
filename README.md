# Live Chat App Server

## Client

-   [Github](https://github.com/PhamCong01/liveChatFrontend)
-   [Deploy](https://pansos.netlify.app)

## Build

```sh
git clone https://github.com/jackdo1012/live-chat-server
cd live-chat-server
yarn
yarn build
yarn start
```
